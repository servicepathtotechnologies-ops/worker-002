Test the workflow builder with simpler prompts first to identify where it's failing:

--------------------------

TEST 3: AI Integration
----------------------

Expected:
- Trigger
- AI Agent

TEST 4: Multiple Integrations
-----------------------------
Prompt: "Save data to Google Sheets and send an email via Gmail"

Expected:
- Trigger

- Gmail

TEST 5: Nested Conditional
--------------------------
Prompt: "If condition A is true, then check if condition B is true, and if both are true, send a message"

Expected:
- Trigger

Use these simpler tests to identify which aspect of the workflow builder is failing.
